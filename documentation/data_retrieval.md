### How to access the datasets that we used

go to kaggle.com and search for the following datasets: 

US Traffic Accidents or go to the link https://www.kaggle.com/datasets/usdot/nhtsa-traffic-fatalities/data

and 

alcohol consumption us or go to the link https://www.kaggle.com/datasets/linzey/alcohol-consumption-us

after downloading those datsets, you will need to extract those files from the zip folder and place them in the raw_data folder located in the root directory of this project.
